# Hub API Endpoints

We have open endpoints that you can use to retrieve information from the Hub as well as perform certain actions such as creating model, dataset or Space repos. We offer a wrapper Python client, `huggingface_hub`, and a JS client, `huggingface.js`, that allow easy access to these endpoints. We also provide webhooks to receive real-time incremental info about repos. Enjoy!

> We’ve moved the Hub API Endpoints documentation to our OpenAPI Playground, which provides a comprehensive reference that’s always up-to-date. You can also access the OpenAPI specification directly at https://huggingface.co/.well-known/openapi.json, or in Markdown version if you want to send it to your Agent: https://huggingface.co/.well-known/openapi.md.

> All API calls are subject to the HF-wide Rate limits. Upgrade your account if you need elevated, large-scale access.
